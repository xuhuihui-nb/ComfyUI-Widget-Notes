# ComfyUI-Widget-Notes
一个轻量级的ComfyUI扩展，用于直接在节点参数（小部件）下添加内联注释
A lightweight ComfyUI extension for adding inline comments directly under node parameters (widgets)

节点和子图可以同步显示和调整

鼠标滚轮可以滚动浏览，ctrl+鼠标滚轮可以整体调整字体大小,目前无法单独调整指定文字的大小.

拖动横线就可以调整文本框的高度

Nodes and subgraphs can be displayed and adjusted synchronously

You can scroll with the mouse wheel, and use Ctrl + mouse wheel to adjust the overall font size (currently, adjusting the size of specific text individually is not supported)

Drag the horizontal line to adjust the height of the text box.
